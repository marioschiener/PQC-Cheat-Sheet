# PQC Cheat Sheet
A cheat sheet for the new post quantum cryptography (PQC) algorithms.

The target audience is technical roles such as developers, architects or engineers. The cheat sheet aims to make it easier for them to select the right algorithm for their use cases, based on various different algorithm characteristics presented in a simplified form.

## Compiling
Use LuaLaTeX for compiling the .tex files. You may need to install the provided fonts in your system first. Set the -shell-escape option in order to reconvert the *.svg graphics after any change during compilation (this requires inkscape to be installed).
